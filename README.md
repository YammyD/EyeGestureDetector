# EyeGestureDetector
 
webcam eye gesture detection using dlib and opencv(only blinking and squinting)
 
# Requirement
 
* python 3.7.9
* opencv == 3.4.5.20
* dlib == 19.21.1
* numpy == 1.19.4
* imutils == 0.5.3
 
# Installation
 i used pip
 
# Usage
 
```bash
git clone https://github.com/YammyD/EyeGestureDetector.git
cd EyeGestureDetector
python main.py
```
